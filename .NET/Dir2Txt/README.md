# Dir2Txt

## 概要
指定フォルダ配下のファイル構成と内容をテキスト化し、テキストから元のフォルダ構成・ファイルを復元できる Windows フォームアプリです。.NET Framework／C#。

## 画面と使い方
- **メイン画面（Directory To Text）**
  - 収集フォルダを指定し「実行」でテキスト化。結果は画面のテキストボックスに表示されます。
  - 収集から除外したい項目を以下で指定できます（いずれも `/` 区切りで複数指定可能、大小文字は無視）。
    - 除外フォルダ: `bin/obj/.git` のようにフォルダ名で一致。
    - 除外ファイル: `Thumbs.db/desktop.ini/readme.txt` のようにファイル名で一致。
    - 除外拡張子: `.log/.tmp` や `log/tmp` のように拡張子を指定（先頭ドット有無どちらでも可）。
  - バイナリと判定したファイルは自動で除外されます。
  - 最終行に改行がない場合は 1 行追加されます。
- **復元画面（復元）**
  - 復元先フォルダと、メイン画面で出力したテキストを指定し「実行」で復元します。
  - 実行前に確認ダイアログ表示。復元先が存在する場合は上書き確認を行い、存在しない場合は新規作成します。

## 出力フォーマット例
```
@@C:\test\hoge\fuga.java|utf-8
1234
@@C:\test\temp\moge.txt|utf-8
abcd
```
`@@` 行に元ファイルのフルパスとエンコード名を記録し、その直後に内容が入ります。

## 設定保存
メイン画面のフォルダパス・除外フォルダ・除外ファイル・除外拡張子は、終了時に実行ファイルと同じ場所の `Dir2Txt.settings.json` に保存され、起動時に読み込まれます。
