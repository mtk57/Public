 SearchShapeForm - 図形内テキスト検索アドイン Readme
============================================================
作成日：2025/06/07
バージョン：1.0

概要
------------------------------------------------------------
このExcelアドインは、シート上の図形（テキストボックス、オートシェイプ、グループ化された図形など）の内部に含まれる文字列を検索するためのツールです。
Excelの標準検索機能（Ctrl+F）では検索できない、フローチャートや構成図などの図形内テキストを素早く見つけ出すことを目的としています。

主な機能
------------------------------------------------------------
* アクティブなシート上の全図形を対象に、内部のテキストを検索します。
* グループ化された図形の中にあるテキストも再帰的に検索します。
* ショートカットキーで検索用のダイアログを起動します。
* Enterキーを押すたびに、次の一致箇所へジャンプします。
* 検索ダイアログを表示したままでも、Excelシートの操作が可能です（モードレス形式）。

インストール手順
------------------------------------------------------------
1.  このアドインファイル「SearchShapeForm.xlam」をPCの任意の場所に保存します。
    ※後からファイルを移動させると設定が切れるため、普段使わないフォルダ（例：ドキュメント内の専用フォルダなど）に保存することを推奨します。
2.  Excelを開き、[ファイル]タブ > [オプション] を選択します。
3.  左のメニューから [アドイン] を選択します。
4.  画面下部にある [管理] ドロップダウンが [Excel アドイン] になっていることを確認し、[設定] ボタンをクリックします。
5.  [アドイン] ダイアログで [参照] ボタンをクリックし、手順1で保存した「SearchShapeForm.xlam」ファイルを選択します。
6.  アドインの一覧に「Searchshapeform」が追加され、チェックボックスにチェックが入っていることを確認して [OK] をクリックします。

以上でインストールは完了です。

使い方
------------------------------------------------------------
1.  Excelファイルを開いている状態で、以下のショートカットキーを押します。
    `Ctrl` + `Shift` + `S`
2.  「図形内テキスト検索」というタイトルのダイアログが表示されます。
3.  テキストボックスに検索したい文字列を入力します。
4.  `Enter` キーを押すと、最初に見つかった図形まで画面がスクロールします。
5.  さらにもう一度 `Enter` キーを押すと、次に見つかった図形へ移動します。
    （一致する最後の図形まで到達すると、次は最初に見つかった図形に戻ります）
6.  検索を終了するには、ダイアログの [終了] ボタンをクリックします。

注意事項
------------------------------------------------------------
* このアドインは、インストールしたPCのExcelでのみ有効です。アドイン自体がExcelファイルに添付されることはありません。SharePointなどで共有されたファイルを他の人が開いても、その人のPCにアドインがなければ、この機能は使えません。
